First steps:

  1. Log in to the Magento server as, or switch to, a user who has permissions to write to the Magento file system. One way to do this is to switch to the web server user.
  2. Change to the following directory:
    cd <your Magento install dir>/bin
    Examples:
    • Ubuntu: cd /var/www/magento2/bin
    • CentOS: cd /var/www/html/magento2/bin

In addition to the arguments discussed in this topic, see Common arguments.
