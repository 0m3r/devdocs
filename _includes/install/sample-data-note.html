If you're installing sample data for a version of Magento 2 earlier than 0.42.0-beta10, you must install the sample-data-media package as well.

Examples:

composer require magento/sample-data:<version> magento/sample-data-media:<version>
where <version> is either an exact version or semantic version syntax.
Exact version example:
composer require magento/sample-data:0.42.0-beta6 magento/sample-data-media:0.42.0-beta1
Next significant release example:
composer require magento/sample-data:~0.42.0-beta5 magento/sample-data-media:~0.42.0-beta1
