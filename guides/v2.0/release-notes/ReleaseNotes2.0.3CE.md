<h2>Magento Community Edition 2.0.3</h2>
Please note that Magento Community Edition 2.0.3 has been replaced by Magento 2.0.4. This release includes all of the security enhancements and performance improvements of Magento 2.0.3, in improved packaging. **You must download and install 2.0.4 to ensure that you receive all the security enhancements of 2.0.3**.
