<h2 id="compman-overview">Overview of the Module Manager</h2>
This section discusses how to start the Module Manager, which updates or uninstalls Magento components:

*	Modules (extend Magento capabilities)
*	Themes (change the look and feel of your storefront and Admin)
*	Language packages (localize the storefront and Admin)

<div class="bs-callout bs-callout-warning">
    <p>If you installed the Magento application by <a href="{{page.baseurl}}install-gde/prereq/dev_install.html">cloning the GitHub repository</a>, you <em>cannot</em> use the Module Manager utility to update components. Instead, you must <a href="{{page.baseurl}}install-gde/install/cli/dev_options.html">update them manually</a>.</p>
</div>
