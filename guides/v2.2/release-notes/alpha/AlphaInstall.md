## Installation 


To install either Magento 2.2.0 Alpha CE and EE, you'll need to: 

Install Magento 2.2.0 CE Alpha Release Candidate

Install Magento 2.2.0 CE Alpha Release Candidate

Install Magento 2.2.0 B2B Alpha Release Candidate
