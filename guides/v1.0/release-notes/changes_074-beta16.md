<h3 id="rn-074b16-changes-join">Join directive</h3>
Created a Join directive, join process for tables, and XML configuration support to define a performant join for search services.

This change also enables you to get a table prefix without injecting `Magento\Framework\App\Resource` into the <a href="{{ site.mage2000url }}lib/internal/Magento/Framework/Api/ExtensionAttributesFactory.php" target="_blank">ExtensionAttributesFactory</a>.
