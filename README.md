How to Contribute
=================

Follow these steps to contribute to this repository:

1. Clone this repository.
2. cd into the clone.
3. Check out a local branch.
4. Do your work.
5. Commit your changes.
6. Push your changes to devbeta: git push origin <local_branch>:devbeta
